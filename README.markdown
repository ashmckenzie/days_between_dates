Days between dates
==================

Determine the number of days, work days and weekend days between two dates.

Example
-------

    ./days_between_dates.rb -s 2010/07/09 -e 2010/08/24

    ===========================================
    Computing between 2010-07-09 and 2010-08-24
    ===========================================
    - Total days: 46
    - Week days: 33
    - Weekend days: 14
